# GPS-Denied Autonomous Navigation

A vision-inertial navigation system that continues estimating position and orientation when GPS is jammed or unavailable.  
The system fuses camera motion with IMU data to generate real-time state estimation and trajectory.

Project Website: [https://aerovision-nine.vercel.app/](https://aerovision-nine.vercel.app/)

> Realtime sensor dashboard runs locally as it requires continuous live data streaming.
